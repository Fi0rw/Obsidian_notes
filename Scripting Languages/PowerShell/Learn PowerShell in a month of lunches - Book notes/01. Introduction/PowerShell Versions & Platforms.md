#### PowerShell Versions
- **PowerShell 5**: La version 5 de PowerShell es una version basada en el framework .NET, es únicamente compatible con sistema operativo Windows. El nombre del proceso es `powershell.exe`. Esta versión se encuentra limitada en la sintaxis de comandos y generalmente se encuentra instalada por defecto.
- **PowerShell 7**: La version 7 de PowerShell es una versión basada en .NET framework CORE. Se instala en paralelo (side-by-side) e introduce nuevas características en la sintaxis como `&&`, etc. El nombre del proceso es `pwsh.exe`, este cambio se realizó para distinguir entre la versión tradicional que sigue siendo `powershell.exe` (versión 5).

#### PowerShell platforms
- Es posible usar PowerShell diferentes plataformas como: Windows, Linux, MacOS, Azure, Windows Server, y containers. 
- **PowerShell on Windows**: Hay multiples formas de instalar PowerShell version 7 en Windows, es posible hacerlo desde la Microsoft Store, con winget, Chocolatey, desde el repositorio de GitHub, etc.    
- **PowerShell on Linux**: Es posible instalar PowerShell en distintas distros de Linux como Ubuntu, Debian, Fedora, CentOS, Arch Linux, etc. La instalación puede variar entre distribuciones.
