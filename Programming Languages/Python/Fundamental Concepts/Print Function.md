Cada lenguaje de programación tiene alguna forma de mostrar datos en la terminal con un método incorporado, función, propiedad o palabra clave. En Python es posible usar la función `print` para imprimir datos en la terminal.

~~~Python
print('Hello World') # Esto imprime la palabra "Hello World"
~~~

Es posible usar la función `print` para mostrar múltiples valores o argumentos a la vez separandolos por comas (tener en cuenta que Python agrega un espacio entre cada elemento cuando se separan por comas). E.g: 

~~~Python
print('My favorite colors are', 'green', 'white', 'purple')

# Output: My favorite colors are green white purple
~~~
